Webhooks allow external services to be notified when certain events happen. 
When the specified events happen, we'll send a **POST** request to each of the URLs you provide.

- https://jenkins.ma-backbone.net/github-webhook/ (pull_request, pull_request_review, pull_request_review_comment, and push)

- https://www.notion.so/eap/webhook/8bfe6bb0-ed50-4e1b-9470-09979a9e64d0 (pull_request)
- https://api.eu.snyk.io/webhook/github-cloud-app/5814914d-3461-4ac9-98bf-ac514514013f (issue_comment, pull_request, pull_request_review_comment, and push)
